Micro-Doppler

FallSafe uses advanced technology to detect micro-doppler shifts associated with falls. Using machine learning algorithms it can determine if a fall has occured.

Read more

Machine Learning

AI determines if a fall has occurred. A softmax regression classifier is paired with stacked autoencoders for optimized detection, based on recent work by Jokanovic et al.[REF].

Read more

Emergency Contacts

Emergency contacts can be contacted in the event of a fall. In any case, emergency services will be contacted if FallSafe is certain that there was a serious fall.

Read more

Privacy

FallSafe cannot capture images, only the speed of movements, making it safe to use in private areas - unlike a camera. Falls can occur anywhere in the home, including the bedroom and bathroom.
